API de Status de Bateria.

A API do status da bateria, mais frequentemente referida como a API de bateria, fornece informações sobre o nível de carga da bateria do sistema e permite que o usuário seja notificado por eventos que são enviados quando ocorrem mudanças de nível da bateria. Isso pode ser usado para ajustar o uso de recursos de seu aplicativo para reduzir o consumo da bateria quando a bateria estiver fraca, ou para salvar as alterações antes que a bateria acabe, a fim de evitar a perda de dados.
